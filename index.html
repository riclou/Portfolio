Skills

HTML (Hypertext Markup Language): É uma linguagem de marcação usada para criar páginas web. É a espinha dorsal de qualquer site e é usada para estruturar o conteúdo e definir elementos como títulos, parágrafos, imagens, links, formulários, entre outros.

CSS (Cascading Style Sheets): É uma linguagem de estilo usada para controlar a aparência de páginas web. É usada para definir a aparência de elementos HTML, como cores, fontes, margens, tamanhos, layout e efeitos visuais.

JavaScript: É uma linguagem de programação amplamente usada para adicionar interatividade e funcionalidade dinâmica a páginas web. É uma linguagem de script que permite a manipulação de elementos HTML, interação do usuário, animações, validação de formulários, acesso a APIs, entre outras funcionalidades.

Git: É um sistema de controle de versão distribuído amplamente utilizado para o gerenciamento de código fonte. Ele permite que os desenvolvedores acompanhem as mudanças em seus arquivos de código, trabalhem em equipe, mesclam e compartilhem código de forma eficiente e revertam alterações se necessário.

React: É uma biblioteca de JavaScript para a criação de interfaces de usuário (UI) interativas e reativas. É uma ferramenta popular para o desenvolvimento de aplicações web de página única (SPA), onde as atualizações na interface do usuário são feitas de forma eficiente sem recarregar a página inteira.

Node.js: É um ambiente de tempo de execução JavaScript do lado do servidor, construído sobre o motor de JavaScript V8 do Google Chrome. Ele permite que os desenvolvedores usem JavaScript para criar aplicativos de servidor, lidar com solicitações HTTP, gerenciar arquivos, bancos de dados, realizar operações assíncronas e muito mais.

Bootstrap: É um framework de código aberto amplamente usado para o desenvolvimento de interfaces web responsivas e estilizadas. Ele fornece uma coleção de estilos, componentes e utilitários pré-projetados que facilitam a criação de páginas web modernas e visualmente atraentes.
